Outlook Add-in Installation Guide

To install the Email AI Assistant in Outlook, follow these steps:

  1. Click the button below to download the manifest file:

    Direct Download

  2. Outlook Add-in Link

    Launch Outlook and click Get Add-ins or My Add-ins from the Main Menu.

  3. In Manage My Add-ins settings, select Custom Add-ins.

  4. Select the Add from File option and choose the downloaded manifest.xml file.

    Note: Microsoft has removed the "Add from URL" option, so installation is only possible via file method.

  5. Click Install to complete the add-in installation.

This application needs to run continuously in the background while Outlook is running.

You can enable the auto-start option at Windows startup for convenient use.

About Email AI Assistant

Email AI Assistant is a tool that uses artificial intelligence to analyze emails in Outlook and generate automatic replies.

Key Features:

  • Automatic email content summarization
  • Automatic reply generation in various tones (professional, friendly, formal, casual)
  • One-click reply application
  • Runs in background for convenient use

How to Use:

  1. Set up your API key.
  2. Install the add-in in Outlook.
  3. Open an email and check the automatically generated summary and reply in the add-in panel.
  4. Click the "Apply to Reply" button to use the automatically generated reply.
